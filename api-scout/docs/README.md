# API Scout MCP Server Documentation

Welcome to the documentation for **API Scout**, an MCP server designed to revolutionize how you test and explore APIs directly within your AI-assisted workflows.

## Overview

API Scout provides a comprehensive suite of tools to:
- **Manage API Documentation**: Seamlessly import OpenAPI/Swagger specifications.
- **Smart Caching**: Automatically detect changes in your API docs using hash comparisons.
- **Deep Search**: Find endpoints by path, summary, description, or tags.
- **Secure Credentials**: Manage API keys, tokens, and OAuth credentials securely.
- **Execute Calls**: Make authenticated API requests to whitelisted URLs.
